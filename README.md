# Detector de Resolução de Tela em Java Swing

Projeto desenvolvido durante meus estudos de Java e Swing para identificar e exibir a resolução da tela do computador.

## Conceitos abordados

* Classe `Toolkit`
* Obtenção da resolução da tela
* Interface gráfica com Java Swing
* Eventos
* Manipulação de componentes visuais

## Tecnologias utilizadas

* Java
* Java Swing
* NetBeans

## Funcionalidades

* Detecta automaticamente a resolução da tela do computador.
* Exibe a largura e a altura da tela em uma interface gráfica.
* Utiliza elementos gráficos para melhorar a experiência visual.

## Como executar

1. Clone o repositório.
2. Abra o projeto no NetBeans.
3. Execute a classe principal.

## Objetivo

Aprender a utilizar recursos do sistema operacional através da API Java e desenvolver aplicações desktop utilizando Java Swing.
